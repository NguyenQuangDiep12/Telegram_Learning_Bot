🔧 FIX LỖI DEPLOY TRÊN RENDER
=====================================

VẤN ĐỀ:
-------
Render đang dùng Python 3.13 không tương thích với python-telegram-bot

GIẢI PHÁP:
---------

1. Trong Render Dashboard:
   - Vào service settings
   - Tìm mục "Python Version" hoặc "Build & Deploy"
   - Chọn "Auto" hoặc "Python 3.11"
   - Save changes

2. HOẶC force trong Build Command:
   Build Command:
   pyenv install 3.11.0 && pyenv global 3.11.0 && pip install -r requirements.txt

3. KIỂM TRA:
   - Xem logs trên Render
   - Phải thấy: "Running with Python 3.11"

CÁC THAY ĐỔI ĐÃ LÀM:
------------------
✅ requirements.txt: python-telegram-bot==20.8
✅ runtime.txt: 3.11.0
✅ config.py: Thêm OPENAI_API_KEY
✅ Dockerfile: Force Python 3.11

DEPLOY:
-------
git add .
git commit -m "Fix Python version compatibility"
git push
